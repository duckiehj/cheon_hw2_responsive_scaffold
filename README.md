# HW2 Responsive Scaffold
This repo contains the submission for 'MMED-1055 Assignment#2 Responsive Scaffold' by Jin Cheon.  
The Project focuses on building a single responsive web page that adapts seamlessly to mobile, tablet, and desktop screen sizes using CSS media queries. The layout is developed with a mobile-first approach.
